# Calligrid

Calligrid is a calligraphy grid generator built with React, TypeScript, and Vite.

## Features

- Generate A4 and B5 calligraphy practice sheets
- Configure cell size and row/column counts
- Toggle multiple guide overlays such as cross, diagonal, nine-square, and well-circle
- Switch grid line color between red and black
- Preview a single cell and a full page layout
- Export the current layout as PDF

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

GitHub Actions workflow is configured for GitHub Pages deployment.

Target URL:

`https://minsecrus.github.io/Minsecrus/CalliGrid`

In the repository settings, set Pages source to `GitHub Actions`.
